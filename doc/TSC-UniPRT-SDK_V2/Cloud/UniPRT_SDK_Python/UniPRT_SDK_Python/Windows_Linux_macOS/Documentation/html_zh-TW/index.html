在 Visual Studio Code 中匯入 SDK

  1. 將 SDK 套件下載到您的本機目錄。
  2. 建立一個 Visual Studio Code 專案。
  3. 將 UniPRT_Cloud-1.0.0-py3-none-any.whl 加入到專案檔案中。
  4. 使用指令建立虛擬環境：
    python -m venv ENVNAME
  5. 使用指令啟動虛擬環境：
    linux/macOS:
    source myenv/bin/activate
    PowerShell(Windows)：
    .\ENVNAME\Scripts\Activate
    CMD(Windows):
    ENVNAME\Scripts\activate.bat
  6. 使用指令安裝 SDK 套件：
    pip install UniPRT_Cloud-1.0.0-py3-none-any.whl
  7. 在專案中匯入 MQTTSDK：
    from UniPRT_Cloud.Comm.MqttComm import MqttComm
    UniPRT_Cloud.Comm.MqttComm
    Definition MqttComm.py:1

TSC Topic 字首設定

此 Topic 結構允許使用者自訂字首，以便更靈活地進行整合。

Topic 結構

Topic 遵循以下格式：

{Tenant ID}/{Topic Prefix}/{Topic Subject}/{Agent ID}
備註
若要連線至 TSC CloudConnect，Topic需設置為 TSCCloudConnect

其中 Topic Subject 包含：

  • request/json
  • status/json
  • response/json
  • alert/json

範例

若某租戶的 ID 為 02bfda5a55d15f29214b99287a4e623f，使用的 Topic 字首為 CloudConnect，且 Topic 內容為 status/json，裝置為 MH241-ABC-123，則完整 Topic 為：

02bfda5a55d15f29214b99287a4e623f/CloudConnect/clients/status/json/MH241-ABC-123
