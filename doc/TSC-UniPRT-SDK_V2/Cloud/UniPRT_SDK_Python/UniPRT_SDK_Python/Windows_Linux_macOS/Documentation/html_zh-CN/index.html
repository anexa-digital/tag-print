在 Visual Studio Code 中导入 SDK

  1. 将 SDK 包下载到您的本地目录。
  2. 创建一个 Visual Studio Code 项目。
  3. 将 UniPRT_Cloud-1.0.0-py3-none-any.whl 添加到项目文件。
  4. 使用命令创建虚拟环境：
    python -m venv ENVNAME
  5. 使用命令激活虚拟环境： linux/macOS:
    source myenv/bin/activate
    PowerShell(Windows)：
    .\ENVNAME\Scripts\Activate
    CMD(Windows):
    ENVNAME\Scripts\activate.bat
  6. 使用命令安装 SDK 包：
    pip install UniPRT_Cloud-1.0.0-py3-none-any.whl
  7. 在项目中导入 MQTTSDK：
    from UniPRT_Cloud.Comm.MqttComm import MqttComm

TSC Topic 前缀配置

此 Topic 结构允许用户自定义前缀，以便更灵活地进行集成。

Topic 结构

Topic 遵循以下格式：

{Tenant ID}/{Topic Prefix}/{Topic Subject}/{Agent ID}
备注
若要连接至 TSC CloudConnect，Topic 必须设置为 TSCCloudConnect。

其中 Topic Subject 包含：

  • request/json
  • status/json
  • response/json
  • alert/json

示例

如果某租户的 ID 为 02bfda5a55d15f29214b99287a4e623f，使用的 Topic 前缀为 CloudConnect，且 Topic 内容为 status/json，设备为 MH241-ABC-123，则完整 Topic 为：

02bfda5a55d15f29214b99287a4e623f/CloudConnect/clients/status/json/MH241-ABC-123
