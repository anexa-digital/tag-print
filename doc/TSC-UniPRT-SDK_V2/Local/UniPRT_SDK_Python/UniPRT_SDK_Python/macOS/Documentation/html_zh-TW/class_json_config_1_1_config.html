◆ get_name()

JsonConfig.Config.get_name ( self)

獲取與配置關聯的名稱。

配置名稱。如果不需要命名，該屬性可以留空。

該屬性可用於為配置提供更具描述性的名稱，例如“ShipLabel”。

請注意，名稱應保持簡短，以防止被打印機截斷。

傳回值
與配置關聯的名稱。

◆ get_number()

JsonConfig.Config.get_number ( self)

獲取與配置關聯的編號。

配置編號。有效範圍為0-8。0表示出廠配置，1-8表示用戶存儲的配置。

此編號指的是存儲在打印機內存中的配置編號。

傳回值
與配置關聯的編號。

◆ get_program_file()

JsonConfig.Config.get_program_file ( self)

獲取與配置關聯的程序文件。

可以將配置下載到的打印機程序文件。

如果留空，打印機將不會阻止將配置下載到不同的程序文件。

如果您想確保配置僅應用於具有匹配程序文件的打印機，請保留從打印機讀取的此值。

傳回值
與配置關聯的程序文件。
