必須傳遞一個打開的TcpComm或UsbComm連接，如果監聽來自打印機的未請求消息，該連接可以保持打開狀態。

發送/接收打印機狀態消息需要一個連接。

參數
connectionTCP或USB通信對象。
mgmt_commJSON通信對象。
comm_descriptor通信描述符。
傳回值
一個初始化的SettingsReadWrite實例，如果對象無法創建則返回nil。
