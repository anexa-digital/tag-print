◆ init_with_comm_descriptor()

OdvMonitor.OdvMonitor.init_with_comm_descriptor ( self,
comm_descriptor )

創建一個由SDK管理的連接時調用的構造函數。

commDescriptor是用於連接打印機的通信描述符。

TCP描述符格式："ip" 或 "ip:port"

例如："127.0.0.1" 或 "127.0.0.1:3007"

如果未提供端口，則使用默認值。

參數
comm_descriptor通信描述符。
傳回值
一個初始化的OdvMonitor實例，如果無法創建該對象則返回nil。
