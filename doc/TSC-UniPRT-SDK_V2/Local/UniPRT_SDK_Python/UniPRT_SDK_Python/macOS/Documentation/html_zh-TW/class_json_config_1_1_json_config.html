必須傳遞一個已打開的 TcpComm 或 UsbComm 連接，如果要監聽來自打印機的非請求消息，該連接可以保持打開狀態。更多...

參數
connectionTCP 或 USB 通信對象。
mgmt_commJSON 通信對象。
comm_descriptor用於連接打印機的通信描述符，TCP 描述符格式："ip" 或 "ip:port"，例如 "127.0.0.1" 或 "127.0.0.1:3007"。如果未提供端口，則使用默認值。
傳回值
一個已初始化的 JsonConfig 實例，如果無法創建對象，則為 nil。
