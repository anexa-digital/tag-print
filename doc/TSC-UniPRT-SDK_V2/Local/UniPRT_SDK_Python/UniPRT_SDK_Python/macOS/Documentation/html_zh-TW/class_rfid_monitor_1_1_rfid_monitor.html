RfidMonitor.RfidMonitor.init_with_comm_descriptor ( self,
comm_descriptor )

創建由SDK管理的連接時調用的構造函數。

commDescriptor 是用於與打印機連接的通信描述符。

TCP描述符格式："ip" 或 "ip:port"

例如 "127.0.0.1" 或 "127.0.0.1:3007"

如果沒有提供端口，則使用默認值。

參數
commDescriptor通信描述符。
傳回值
一個初始化的RfidMonitor實例，如果對象無法創建則返回nil。
