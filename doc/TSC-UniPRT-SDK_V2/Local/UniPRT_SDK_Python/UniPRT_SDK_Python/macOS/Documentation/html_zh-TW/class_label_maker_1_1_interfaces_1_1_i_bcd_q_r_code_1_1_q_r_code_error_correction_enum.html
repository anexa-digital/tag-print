詳細描述

錯誤校正允許條碼在損壞的情況下仍然可讀，但會增加編碼的數據量。

更高的錯誤校正允許條碼在受到更多損壞的情況下仍然可讀。 由於在更高的錯誤校正級別使用了更多的數據，條碼的大小/面積會增加。
