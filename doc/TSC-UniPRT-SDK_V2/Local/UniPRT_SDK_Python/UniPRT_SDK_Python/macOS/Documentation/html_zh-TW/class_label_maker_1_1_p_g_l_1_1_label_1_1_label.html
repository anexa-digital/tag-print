詳細描述

表示可以格式化打印的標籤。

此類提供用於創建和管理具有指定設置的標籤的功能，允許添加對象和原始內容，以便格式化為打印機語言語法。

向標籤添加原始內容，以便在需要添加自定義內容的情況下使用。

添加原始內容允許添加可能不可用的打印機語言命令或其他功能。不進行內容格式化；調用者負責確保內容遵循打印機語言語法。
