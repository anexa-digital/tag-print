Maxicode 模式。

Maxicode 條碼有兩個主要的消息結構，稱為主消息和次消息。模式決定了消息結構的內容和錯誤校正（EEC = 增強型錯誤校正，SEC = 標準錯誤校正）。

由於每個消息部分（主消息或次消息）可以具有不同的錯誤校正級別，因此要編碼到Maxicode條碼中的數據根據模式劃分為消息部分。

  • 模式 2: 結構化承載消息：6-9 位郵政編碼（數字）。
  • 模式 3: 結構化承載消息：6 位郵政編碼（字母數字）。
  • 模式 4: 標準符號。非結構化數據，通用條碼。EEC 主消息，SEC 次消息。
