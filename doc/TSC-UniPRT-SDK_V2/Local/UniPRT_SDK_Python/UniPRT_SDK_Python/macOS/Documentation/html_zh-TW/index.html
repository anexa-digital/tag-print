概述

我們的SDK提供了一套全面的工具，幫助您將我們的功能集成到您的應用程序中。請閱讀我們的示例代碼，您將了解最重要的類以及SDK的運作原理。

  • TestMgmtCommSDK: 有關如何讀取打印機配置的JSON消息的示例代碼。
  • TestPrinterMonitor: 有關如何讀取打印機信息的JSON消息的示例代碼。
  • TestRfidMonitor: 有關如何持續接收來自打印機的RFID消息的示例代碼。
  • TestOdvMonitor: 有關如何持續接收來自打印機的ODV消息的示例代碼。
  • TestLabelMaker: 有關如何構建標籤的示例代碼，包括TSPL和PGL。
  • TestPrinterDiscovery: 有關如何搜尋打印機的示例代碼。

將SDK導入Visual Studio Code

  1. 下載SDK包到您的PC本地目錄。
  2. 使用以下命令下載Homebrew：
    /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
  3. 使用以下命令安裝libusb包：
    brew install libusb
  4. 創建一個Visual Studio Code項目。
  5. 將UniPRT_python-2.0.0.0-cp313-cp313-macosx_15_0_arm64.whl添加到項目文件中。
  6. 使用以下命令創建環境：
    python -m venv ENVNAME
  7. 使用以下命令激活環境：
    source myenv/bin/activate
  8. 使用以下命令安裝包：
    pip install UniPRT_python-2.0.0.0-cp313-cp313-macosx_15_0_arm64.whl
  9. 將UniPRT導入到您的項目中：
    from UniPRT.xxx.xxx import xxx
