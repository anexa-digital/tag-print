Detailed Description

RFID memory block selection.

  • INVALID_MEM_BLOCK: Invalid memory block for initialization.
  • EPC: Electronic Product Code.
  • TID: Tag Identification (read-only).
  • USER: User memory.
  • ACCESS_CODE: Access code (typically 4 bytes).
  • KILL_CODE: Kill code (typically 4 bytes).
  • PC: Protocol Control Word (16 bits, Gen 2 tags only).
  • RESERVE: Reserved memory.
