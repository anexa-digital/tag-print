详细描述

表示 PDF417 条形码的类。

此类继承自 APdf417，并提供了用于初始化、配置和描述 PDF417 条形码的功能。 PDF417 条形码是一种二维条形码格式，用于编码大量数据，广泛用于运输、识别和库存管理等应用中。
