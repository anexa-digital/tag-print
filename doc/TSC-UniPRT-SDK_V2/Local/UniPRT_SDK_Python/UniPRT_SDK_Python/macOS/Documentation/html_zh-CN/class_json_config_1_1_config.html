◆ get_name()

JsonConfig.Config.get_name ( self)

获取与配置关联的名称。

配置名称。如果不需要命名，该属性可以留空。

该属性可用于为配置提供更具描述性的名称，例如“ShipLabel”。

请注意，名称应保持简短，以防止被打印机截断。

返回
与配置关联的名称。

◆ get_number()

JsonConfig.Config.get_number ( self)

获取与配置关联的编号。

配置编号。有效范围为0-8。0表示出厂配置，1-8表示用户存储的配置。

此编号指的是存储在打印机内存中的配置编号。

返回
与配置关联的编号。

◆ get_program_file()

JsonConfig.Config.get_program_file ( self)

获取与配置关联的程序文件。

可以将配置下载到的打印机程序文件。

如果留空，打印机将不会阻止将配置下载到不同的程序文件。

如果您想确保配置仅应用于具有匹配程序文件的打印机，请保留从打印机读取的此值。

返回
与配置关联的程序文件。
