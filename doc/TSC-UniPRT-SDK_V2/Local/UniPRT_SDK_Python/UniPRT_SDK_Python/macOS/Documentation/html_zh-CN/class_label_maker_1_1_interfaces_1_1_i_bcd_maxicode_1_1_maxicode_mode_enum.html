Maxicode 模式。

Maxicode 条形码有两个主要的消息结构，称为主消息和次消息。模式决定了消息结构的内容和错误校正（EEC = 增强型错误校正，SEC = 标准错误校正）。

由于每个消息部分（主消息或次消息）可以具有不同的错误校正级别，因此要编码到Maxicode条形码中的数据根据模式划分为消息部分。

  • 模式 2: 结构化承载消息：6-9 位邮政编码（数字）。
  • 模式 3: 结构化承载消息：6 位邮政编码（字母数字）。
  • 模式 4: 标准符号。非结构化数据，通用条形码。EEC 主消息，SEC 次消息。
