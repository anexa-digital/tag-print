概述

我们的SDK提供了一套全面的工具，帮助您将我们的功能集成到您的应用程序中。请阅读我们的示例代码，您将了解最重要的类以及SDK的运作原理。

  • TestMgmtCommSDK: 有关如何读取打印机配置的JSON消息的示例代码。
  • TestPrinterMonitor: 有关如何读取打印机信息的JSON消息的示例代码。
  • TestRfidMonitor: 有关如何持续接收来自打印机的RFID消息的示例代码。
  • TestOdvMonitor: 有关如何持续接收来自打印机的ODV消息的示例代码。
  • TestLabelMaker: 有关如何构建标签的示例代码，包括TSPL和PGL。
  • TestPrinterDiscovery: 有关如何搜寻打印机的示例代码。

将SDK导入Visual Studio Code

  1. 下载SDK包到您的PC本地目录。
  2. 使用以下命令下载Homebrew：
    /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
  3. 使用以下命令安装libusb包：
    brew install libusb
  4. 创建一个Visual Studio Code项目。
  5. 将UniPRT_python-2.0.0.0-cp313-cp313-macosx_15_0_arm64.whl添加到项目文件中。
  6. 使用以下命令创建环境：
    python -m venv ENVNAME
  7. 使用以下命令激活环境：
    source myenv/bin/activate
  8. 使用以下命令安装包：
    pip install UniPRT_python-2.0.0.0-cp313-cp313-macosx_15_0_arm64.whl
  9. 将UniPRT导入到您的项目中：
    from UniPRT.xxx.xxx import xxx
