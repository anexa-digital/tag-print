详细描述

用于访问/保护支持此功能的标签上的存储区域的密码。

  • NONE：无密码。
  • LOCK：如果提供正确的密码，允许将来修改数据。
  • PERMA_LOCK：防止进一步修改数据。
  • UNLOCK：解锁块，以便可以覆盖它。
  • PERMA_UNLOCK：不能被锁定。
