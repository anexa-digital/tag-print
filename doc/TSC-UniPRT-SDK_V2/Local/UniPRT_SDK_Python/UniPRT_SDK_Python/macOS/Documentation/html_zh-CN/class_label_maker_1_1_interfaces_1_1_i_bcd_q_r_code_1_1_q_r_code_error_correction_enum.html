详细描述

错误校正允许条形码在损坏的情况下仍然可读，但会增加编码的数据量。

更高的错误校正允许条形码在受到更多损坏的情况下仍然可读。 由于在更高的错误校正级别使用了更多的数据，条形码的大小/面积会增加。
