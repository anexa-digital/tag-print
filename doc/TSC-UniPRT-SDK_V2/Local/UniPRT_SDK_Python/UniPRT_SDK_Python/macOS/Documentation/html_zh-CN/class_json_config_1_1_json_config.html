必须传递一个已打开的 TcpComm 或 UsbComm 连接，如果要监听来自打印机的非请求消息，该连接可以保持打开状态。更多...

参数
connectionTCP 或 USB 通信对象。
mgmt_commJSON 通信对象。
comm_descriptor用于连接打印机的通信描述符，TCP 描述符格式："ip" 或 "ip:port"，例如 "127.0.0.1" 或 "127.0.0.1:3007"。如果未提供端口，则使用默认值。
返回
一个已初始化的 JsonConfig 实例，如果无法创建对象，则为 nil。
