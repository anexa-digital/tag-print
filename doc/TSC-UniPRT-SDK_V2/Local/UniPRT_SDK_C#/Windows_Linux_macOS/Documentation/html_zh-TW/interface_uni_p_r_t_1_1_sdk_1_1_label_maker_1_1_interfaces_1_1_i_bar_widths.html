詳細描述

設定窄/寬條的寬度。

請注意，並非所有的條寬比都能被條碼掃描器讀取。典型的寬:窄比為 2:1、2.5:1、3:1、4:1。 在大批量列印之前，應列印一些測試標籤並由預期的條碼掃描器讀取。

◆ Ruler

IRuler UniPRT.Sdk.LabelMaker.Interfaces.IBarWidths.Ruler
getset

用於條寬的標尺。

在某些情況下，可能需要以原生點來指定條寬，以避免捨入錯誤，因為熱感印表機上的最小列印單位是點，任何導致小數點列印的條寬都會導致捨入錯誤。 請注意，捨入錯誤會更嚴重地影響小 X 維度/（NarrowBar）條碼的可讀性，因為捨入錯誤導致 X 維度變小時誤差比例變大。
