詳細描述

支援方法以協助將非字節數據轉換為儲存在 RFID 標籤中的順序。

數據以從 MSB 到 LSB 的順序存儲於 RFID 標籤中。運行 SDK 的 PC 可能是小端或大端，這些支援方法有助於處理可能受運行 SDK 平台的端序影響的轉換。
