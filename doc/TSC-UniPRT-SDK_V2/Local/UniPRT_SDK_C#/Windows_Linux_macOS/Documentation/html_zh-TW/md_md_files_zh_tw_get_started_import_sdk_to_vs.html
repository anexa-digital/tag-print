UniPRT SDK  v2.0.0.3
將 SDK 匯入 Visual Studio

本地 NuGet 套件：使用 NuGet 套件管理器將 SDK 添加到 C# Visual Studio 專案

  • 1. 下載/複製 NuGet 套件（*.nupkg）到您電腦上的某個本地目錄。
  • 2. 建立一個 MS Visual Studio 專案。
    • a. 在 C# 程式碼檔案的頂部添加「using」語句以引用庫。
      例如： 「using UniPRT.Sdk.Comm;」
  • 3. 專案建立後，將 NuGet 套件添加到 MS Visual Studio 專案
    • a. 打開您的 Visual Studio 專案。
    • b. 點擊 專案/管理 NuGet 套件... 選單
      • 或在「方案總管」窗格中右鍵點擊專案，並選擇 管理 NuGet 套件...
    • c. 在 Visual Studio 中添加本地引用至 NuGet 套件（*.nupkg）所在的目錄：
      • i. 預設情況下，Visual Studio 嘗試使用 nuget.org 作為 NuGet 套件的「來源」。我們需要通過點擊 NuGet 套件管理器右上方的「齒輪」來添加本地來源。
      • ii. 當 NuGet 套件管理器的「選項」視窗彈出後，使用「+」按鈕添加包含 NuGet 套件的本地目錄，並為其指定「名稱」和「來源」（目錄或 URL）。
  • 4. 從新增的本地目錄下拉清單中選擇您的 NuGet 套件。如果未顯示該套件，請勾選「包括預發行版本」複選框。
  • 5. 通過選擇套件名稱並點擊「安裝」按鈕來安裝套件。
  • 6. 套件安裝完成後，顯示缺少庫的所有紅色波浪線（錯誤）應該消失，並且 Visual Studio 的「intellisense」自動完成應該可用於庫類別的方法/屬性。
  • 7. 確認 SDK NuGet 套件已添加到專案
    • 在 Visual Studio 方案總管（Visual Studio 右側窗格）
    • SDK 項目應顯示在 方案總管/<projectName>/Dependencies/Packages/_
