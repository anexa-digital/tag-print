詳細描述

印表機配置的容器。

印表機可以保存多個配置。
讀取/寫入不同的配置可以通過使用 Number 屬性完成。

屬性(property)說明文件

◆ Data

string UniPRT.Sdk.Settings.Config.Data
get

印表機配置值。這些值不應被修改。

這些值不應被修改，應僅供閱讀、備份和重新下載到印表機。

◆ Model

string UniPRT.Sdk.Settings.Config.Model
getset

配置適用的印表機型號。

此配置僅能下載到與型號匹配的印表機上。

◆ Name

string UniPRT.Sdk.Settings.Config.Name
getset

配置名稱。如果不需要命名，此屬性可以留空。

該屬性可用於為配置賦予更具描述性的名稱。例如 "ShipLabel"
注意名稱應保持簡短，以防止印表機截斷。

◆ Number

int UniPRT.Sdk.Settings.Config.Number
getset

配置編號。0-8 有效。0 是出廠配置，1-8 是用戶存儲的配置。

此數字指的是存儲在印表機記憶體中的配置編號。

◆ ProgramFile

string UniPRT.Sdk.Settings.Config.ProgramFile
getset

配置可下載到的印表機程式文件。

如果留空，印表機不會阻止將配置下載到不同的程式文件。
如果要確保配置僅應用於與程式文件匹配的印表機，請保留從印表機讀取的值。
