字元大小可選擇使用字型點數或其他大小單位。

X/Y 字元大小以字型點數計算，1 點 = 1/72 英吋（約 0.35 毫米）。
請注意，有些字型可能無法縮放。指定在最小/最大範圍外的任何值將被限制在最小或最大值。
