◆ AlertStatusNotice()

delegate void UniPRT.Sdk.Monitor.AlertStatusNotice ( string[]  alert)

用於設置故障/警報變更通知回調的委派函數簽名類型。

傳入的 alert 字串陣列表示印表機發送的警報號碼在 alert[0] 和簡短描述在 alert[1]。
例如 "2418" 代表 "列印頭打開" 警報。 alert[0] = "2418"; alert[1] = "列印頭打開";

◆ DisplayStatusNotice()

delegate void UniPRT.Sdk.Monitor.DisplayStatusNotice ( string[]  displayLines)

用於設置前面板變更通知回調的委派函數簽名類型。

displayLines 是一個字串陣列，每個條目代表顯示器上的一行文字。
例如，displayLines[0] = 顯示的第一行文字，displayLines[1] = 顯示的第二行文字。

◆ EngineStatusNotice()

delegate void UniPRT.Sdk.Monitor.EngineStatusNotice ( string  engineState)

用於設置引擎狀態變更通知回調的委派函數簽名類型。

傳入的 engineState 是 "fault"、"idle"、"offline"、"pause"、"printing" 或 "present" (標籤在場) 之一。

◆ OdvReportNotice()

delegate void UniPRT.Sdk.Monitor.OdvReportNotice ( OdvReport  odvReport)

用於設置條碼報告接收時回調的委派函數簽名類型。用於設置 OdvReportCallback。

傳遞給函數的參數是代表接收到的 ODV 報告的 OdvReport 物件。
