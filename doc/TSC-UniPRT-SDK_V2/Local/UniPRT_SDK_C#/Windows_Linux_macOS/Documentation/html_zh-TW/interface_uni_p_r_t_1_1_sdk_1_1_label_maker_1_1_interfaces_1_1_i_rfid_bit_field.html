數據的格式。應注意，大多數數據是十六進位格式。

RFID 數據欄位。從最高有效位（MSB）到最低有效位（LSB）（由左至右）。十六進位格式數據。

◆ OffsetFromStart

int UniPRT.Sdk.LabelMaker.Interfaces.IRfidBitField.OffsetFromStart
getset

指定要訪問的記憶體區塊中的位，從 OffsetFromStart 開始。

如果訪問的記憶體不是記憶體區塊的起始位置，可選的字偏移量（每次計數 16 位）。

第一個字從偏移 0 開始，第二個字將在偏移 1，依此類推。
注意：
對於 Gen 2 標籤上的 EPC，CRC（字 0）和 PC 位（字 1）不計入偏移部分。 也就是說，偏移 0 是指 PC 位之後的第一個字。
