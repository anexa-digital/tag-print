UniPRT SDK  v2.0.0.3
将 SDK 汇入 Visual Studio

本地 NuGet 套件：使用 NuGet 套件管理器将 SDK 添加到 C# Visual Studio 专案

  • 1. 下载/复制 NuGet 套件（*.nupkg）到您电脑上的某个本地目录。
  • 2. 建立一个 MS Visual Studio 专案。
    • a. 在 C# 程式码档案的顶部添加「using」语句以引用库。
      例如： 「using UniPRT.Sdk.Comm;」
  • 3. 专案建立后，将 NuGet 套件添加到 MS Visual Studio 专案
    • a. 打开您的 Visual Studio 专案。
    • b. 点击 专案/管理 NuGet 套件... 选单
    • 或在「方案总管」窗格中右键点击专案，并选择 管理 NuGet 套件...
    • c. 在 Visual Studio 中添加本地引用至 NuGet 套件（*.nupkg）所在的目录：
    • i. 预设情况下，Visual Studio 尝试使用 nuget.org 作为 NuGet 套件的「来源」。我们需要通过点击 NuGet 套件管理器右上方的「齿轮」来添加本地来源。
    • ii. 当NuGet 套件管理器的「选项」视窗弹出后，使用「+」按钮添加包含NuGet 套件的本地目录，并为其指定「名称」和「来源」（目录或URL）。
  • 4. 从新增的本地目录下拉清单中选择您的 NuGet 套件。如果未显示该套件，请勾选「包括预发行版本」复选框。
  • 5. 通过选择套件名称并点击「安装」按钮来安装套件。
  • 6. 套件安装完成后，显示缺少库的所有红色波浪线（错误）应该消失，并且 Visual Studio 的「intellisense」自动完成应该可用于库类别的方法/属性。
  • 7. 确认 SDK NuGet 套件已添加到专案
    • 在 Visual Studio 方案总管（Visual Studio 右侧窗格）
    • SDK 项目应显示在 方案总管/<projectName>/Dependencies/Packages/_
