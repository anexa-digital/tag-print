◆ GetEngineStatus()

string UniPRT.Sdk.Monitor.IPrinterMonitor.GetEngineStatus ( )

查询印表机的引擎状态。

引擎状态有助于在发送打印工作前检查印表机是否处于可以打印的状态。
返回的 字符串是 "fault"、"idle"、"offline"、"pause"、"printing" 或 "present" (标签在场) 之一。

在 UniPRT.Sdk.Monitor.PrinterMonitor 内被实现.

◆ GetFaultStatus()

string [] UniPRT.Sdk.Monitor.IPrinterMonitor.GetFaultStatus ( )

查询印表机的故障状态。

故障状态用于检查印表机是否处于需要注意的故障状态，或可能阻止处理任何打印工作。
返回的 alert 字符串数组表示印表机发送的警报号码在 alert[0] 和简短描述在 alert[1]。
例如 "2418" 代表 "打印头打开" 警报。 alert[0] = "2418"; alert[1] = "打印头打开";

在 UniPRT.Sdk.Monitor.PrinterMonitor 内被实现.
