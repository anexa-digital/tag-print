详细描述

印表机配置的容器。

印表机可以保存多个配置。
读取/写入不同的配置可以通过使用 Number 属性完成。

属性说明

◆ Data

string UniPRT.Sdk.Settings.Config.Data
get

印表机配置值。这些值不应被修改。

这些值不应被修改，应仅供阅读、备份和重新下载到印表机。

◆ Model

string UniPRT.Sdk.Settings.Config.Model
getset

配置适用的印表机型号。

此配置仅能下载到与型号匹配的印表机上。

◆ Name

string UniPRT.Sdk.Settings.Config.Name
getset

配置名称。如果不需要命名，此属性可以留空。

该属性可用于为配置赋予更具描述性的名称。例如 "ShipLabel"
注意名称应保持简短，以防止印表机截断。

◆ Number

int UniPRT.Sdk.Settings.Config.Number
getset

配置编号。0-8 有效。0 是出厂配置，1-8 是用户存储的配置。

此数字指的是存储在印表机内存中的配置编号。

◆ ProgramFile

string UniPRT.Sdk.Settings.Config.ProgramFile
getset

配置可下载到的印表机程序文件。

如果留空，印表机不会阻止将配置下载到不同的程序文件。
如果要确保配置仅应用于与程序文件匹配的印表机，请保留从印表机读取的值。
