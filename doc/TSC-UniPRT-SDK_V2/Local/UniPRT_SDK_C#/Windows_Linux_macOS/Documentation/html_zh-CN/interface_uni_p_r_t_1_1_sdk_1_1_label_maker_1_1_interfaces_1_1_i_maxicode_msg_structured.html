详细描述

Maxicode 条码数据。Maxicode 自由格式承载讯息。仅适用于模式 2 和 3。

模式 2、3 的“结构化承载讯息”需要邮政编码、国家代码和服务类别的字段。

  • 讯息格式通常编码如下：
    • <primaryMsg><secondaryMsg> = <postalCode>Gs<countryCode>Gs<serviceClass>Gs<secondaryMsg>EOT
  • 邮政编码（6-9 个字符）、国家代码（3 个数字）、服务类别（3 个数字）
    • 模式 2 = 美国：最多 9 位数字的邮政编码（全部为数字）
    • 模式 3 = 加拿大：6 位数字的邮政编码（字母数字） 其中：

Gs: 组分隔符（0x1D）

  • EOT: 传输结束（0x04）
