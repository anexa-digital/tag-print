字符大小可选择使用字体点数或其他大小单位。

X/Y 字符大小以字体点数计算，1 点 = 1/72 英寸（约 0.35 毫米）。
请注意，有些字体可能无法缩放。指定在最小/最大范围外的任何值将被限制在最小或最大值。
