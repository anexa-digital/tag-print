◆ AlertStatusNotice()

delegate void UniPRT.Sdk.Monitor.AlertStatusNotice ( string[]  alert)

用于设置故障/警报变更通知回调的委托函数签名类型。

传入的 alert 字符串数组表示印表机发送的警报号码在 alert[0] 和简短描述在 alert[1]。
例如 "2418" 代表 "打印头打开" 警报。 alert[0] = "2418"; alert[1] = "打印头打开";

◆ DisplayStatusNotice()

delegate void UniPRT.Sdk.Monitor.DisplayStatusNotice ( string[]  displayLines)

用于设置前面板变更通知回调的委托函数签名类型。

displayLines 是一个字符串数组，每个条目代表显示器上的一行文字。
例如，displayLines[0] = 显示的第一行文字，displayLines[1] = 显示的第二行文字。

◆ EngineStatusNotice()

delegate void UniPRT.Sdk.Monitor.EngineStatusNotice ( string  engineState)

用于设置引擎状态变更通知回调的委托函数签名类型。

传入的 engineState 是 "fault"、"idle"、"offline"、"pause"、"printing" 或 "present" (标签在场) 之一。

◆ OdvReportNotice()

用于设置条码报告接收时回调的委托函数签名类型。用于设置 OdvReportCallback。

传递给函数的参数是代表接收到的 ODV 报告的 OdvReport 对象。
