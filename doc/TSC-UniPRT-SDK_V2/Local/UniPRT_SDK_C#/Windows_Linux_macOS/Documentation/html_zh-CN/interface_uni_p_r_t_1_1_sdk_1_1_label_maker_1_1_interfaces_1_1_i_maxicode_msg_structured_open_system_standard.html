详细描述

Maxicode 条码数据。Maxicode 开放系统标准承载讯息。仅适用于模式 2 和 3。


开放系统标准承载讯息格式通常编码为条码，带有 "[)><Rs>01<Gs>" 前缀，如下所示：
"[)>"<Rs>"01"<Gs><yy><PostalCode><Gs><CountryCode><Gs><ServiceClass><Gs><RemainingMsg><Rs><EOT>
其中：

  • yy: 2 个字符的年份（数字）
  • Rs: 记录分隔符（0x1E）
  • Gs: 组分隔符（0x1D）
  • EOT: 传输结束（0x04）
