static IComm UniPRT.Sdk.Comm.Communication.CreateComm ( string  descriptor)
static

根据描述符创建一个通信对象。无法创建时返回空值。

描述符应具有 <type> 前缀。如果未提供前缀，则默认为 "TCP" 类型。
支持的描述符类型：

  • TcpComm
    • <type>:<ip>:<port> 例如 "TCP:127.0.0.1:9001"
    • 如果未提供端口，则使用默认的数据端口。
