详细描述

设置窄/宽条的宽度。

请注意，并非所有的条宽比都能被条码扫描器读取。典型的宽:窄比为 2:1、2.5:1、3:1、4:1。 在大批量打印之前，应打印一些测试标签并由预期的条码扫描器读取。

IRuler UniPRT.Sdk.LabelMaker.Interfaces.IBarWidths.Ruler
getset

用于条宽的标尺。

在某些情况下，可能需要以原生点来指定条宽，以避免舍入错误，因为热感印表机上的最小打印单位是点，任何导致小数点打印的条宽都会导致舍入错误。 请注意，舍入错误会更严重地影响小 X 维度/（NarrowBar）条码的可读性，因为舍入错误导致 X 维度变小时误差比例变大。
