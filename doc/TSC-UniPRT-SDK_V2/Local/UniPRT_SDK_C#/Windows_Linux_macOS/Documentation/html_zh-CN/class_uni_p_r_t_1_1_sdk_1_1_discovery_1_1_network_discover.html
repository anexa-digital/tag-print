static

返回在网络上发现的印表机地址列表。

搜索是通过广播 UDP 讯息进行的，需要将网络设置为允许 UDP 数据包。

参数
[in]brand获取所有印表机或按品牌过滤。
[in]responseTimeoutMs在收到最后一个印表机响应后，停止监听新响应前的超时时间。
