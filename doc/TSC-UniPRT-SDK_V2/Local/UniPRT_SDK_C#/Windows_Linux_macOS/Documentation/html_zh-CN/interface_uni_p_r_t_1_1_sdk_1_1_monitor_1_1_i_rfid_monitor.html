RfidReportNotice UniPRT.Sdk.Monitor.IRfidMonitor.RfidReportCallback
getset

保存接收到 RFID 报告时要调用的函数。函数必须符合 RfidReportNotice 的签名。

传递给函数的参数是代表接收到的 RFID 报告的 OdvReport 对象。

◆ RfidReportListening

bool UniPRT.Sdk.Monitor.IRfidMonitor.RfidReportListening
getset

启用/禁用监听/解析从印表机发送的未请求的 RFID 报告。

请注意，必须启用此功能才能在接收到 RFID 报告时接收任何通知，RfidReportCallback。
